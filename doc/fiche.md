# Fiche technique – React

## Informations générales
- **Nom complet** : React (souvent scindé en deux bibliothèques distinctes pour le web : `react` et `react-dom`)
- **Créateur / Mainteneur** : Meta Platforms, Inc.
- **Date de création** : Non mentionnée dans les sources
  > information à vérifier indépendamment : React a été initialement déployé par Facebook en 2013
- **Dernière version stable** : v19.2
- **Licence** : Non précisée dans les sources
  > Information à vérifier indépendamment : licence MIT

---

## Langages supportés
- JavaScript
- TypeScript (ajoute le typage statique)
- JSX (JavaScript XML permettant d’écrire du HTML dans du JavaScript)

---

## Dépendances et outils principaux
- **React DOM** : indispensable pour le web
- **React Native** : utilisé pour les applications mobiles
- **Babel** : compile le JSX en JavaScript standard
- **Node.js / npm** : gestion des dépendances
- **Vite** : outil de build moderne pour initialiser et optimiser les projets React
